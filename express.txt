What is Express?
-Express is a fast,unopinionated and minimalist web framework for Node.js
-Express is a "Server side" or "back-end" framework.It is not comparable to client-side
 frameworks likes React,Angular & Vue.It can be used combination with those frameworks to 
 built full stack applications

 Why use Express?
 -Makes building web applications with Node.js MUCH easier
 -used for both server rendered apps as well as API/Microservices
 -Extremley light,fast and free
 -full control of request and response
 -by far the most popular node framework
 -Great to use with client side frameworks as it's all javascript

 Express Middleware
 -Middleware functions are functions that have access to the request and
  response object.Express has built in Middleware but middleware alse comes
  from 3rd party packages as well as custom middleware.
  
  *Execute any code
  *Make Changes to the request/response object 
  *End response cycle 
  *Call next middleware in the stack
